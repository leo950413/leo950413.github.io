Implementation of sorting algorithm

Nov 20, 2024

Sorting algorithm

Insertion sort

Steps

  1. Start with the second element (the first is already “sorted”).
  2. Compare the current element with the elements in the sorted portion of the array.
  3. Shift all elements larger than the current element one position to the right.
  4. Insert the current element in its correct position.
  5. Repeat for all elements.

Implementation

void insertionSort(int *arr, int size) {
    for (int i = 1; i < size; i++) {

        int key = arr[i];
        int j = i - 1;
        // move every element forward
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        // plug in the element
        arr[j + 1] = key;
    }
}

Selection algorithm

Steps

  1. Start with the first element of the array.
  2. Find the Minimum: In the unsorted part of the array, find the smallest element.
  3. Swap: Swap the smallest element with the first element of the unsorted part.
  4. Move to the next element and repeat the process for the rest of the array.
  5. Continue until the array is fully sorted.

Implementation

#include<stdio.h>
void selection_sort(int * arr , int size){
    for(int i=1; i<size; i++){

        int min_idx = i - 1;
        int key = arr[i];
        // find where to plug in
        while(arr[min_idx] < arr[i] && min_idx >= 0){
            min_idx --;
        }

        // swap two value
        int temp = arr[min_idx];
        arr[min_idx] = arr[i];
        arr[i] = temp;
    }
}
